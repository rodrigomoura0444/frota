Workshop Tres60 - Operations Management System
Workshop Tres60 is a professional-grade production dashboard designed for automotive workshops and industrial service centers. It serves as a real-time operational layer that synchronizes administrative ERP data with shop-floor execution, optimizing technician allocation and workflow visibility.

🔑 Core Functionalities
Production Control Board: A high-visibility interface for tracking active Service Orders (OS) versus pending queue.

Technician Resource Allocation: Real-time monitoring of staff availability and active assignments.

Priority Management: Automated visual indicators for urgent interventions to minimize lead times.

Automated Notification Engine: Integrated SMTP system for instant job dispatch and completion alerts.

Cross-Platform Reliability: Engineered to handle environment-specific SSL requirements, ensuring stable performance on both Windows and macOS.

🛠️ Technical Specifications
Backend: Python 3.14 / Django 5.x

Frontend: Bootstrap 5, Custom CSS3, JetBrains Mono Typography for industrial readability.

Infrastructure: Initial SQLite development architecture with planned migration to SQL Server (PHC CS Integration).

Security: CSRF-validated endpoints and secure TLS/SSL SMTP handling.

🚀 Installation & Deployment
Prerequisites
Python 3.10 or higher

Virtual Environment (venv)
